Jeu de valeurs regroupant les codes de type de vaccination

References

  • CDA - FR Vaccination

Définition logique (CLD)

Ce jeu de valeur (ValueSet) inclut les codes selon les règles suivantes :

  • Importe tous les codes contenu dans ActSubstanceAdministrationImmunizationCode
  • Importe tous les codes contenu dans JDV Absent Or Unknown Immunization CISIS
